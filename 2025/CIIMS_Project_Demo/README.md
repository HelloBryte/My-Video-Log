# CIIMS Smart Inventory Management System Demo

[![Watch the video](http://img.youtube.com/vi/dJgjQYroZpI/0.jpg)](https://www.youtube.com/watch?v=dJgjQYroZpI)

This is a technical demonstration of an AI-powered desktop inventory management system based on Python.

## Project Overview

CIIMS (Campus Intelligent Inventory Management System) showcases the integration of AI capabilities into traditional desktop applications, providing intelligent office solutions for campus asset management.

## Key Features

- **Desktop Frontend**: Built with PySide6 for cross-platform compatibility
- **Database**: MySQL for reliable data storage
- **Embedded AI Assistant**: Natural language interaction for inventory queries
- **Dual Authentication**: Password and facial recognition support
- **Real-time Inventory Tracking**: Complete digital workflow for borrowing and returning
